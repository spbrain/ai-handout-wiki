# 개인 메모 관리 웹 어플리케이션 제작

** 모든 기능은 AI와 협업해서 진행합니다.**

# 📌 Post-Note 프로젝트 개요

> **실습 목표**: FastAPI + Next.js + Gemini AI를 활용한 풀스택 웹 애플리케이션 개발

---

## 1. 프로젝트 소개

Post-Note는 **포스트잇 스타일의 개인 메모 관리 웹 서비스**입니다.
이 프로젝트를 통해 다음과 같은 풀스택 개발 역량을 실습할 수 있습니다:

- Python FastAPI로 **REST API 백엔드** 구축
- Next.js(React)로 **반응형 프론트엔드** 구축
- JWT 기반 **인증 시스템** 구현
- SQLite3를 활용한 **데이터베이스 설계 및 CRUD**
- Google Gemini API를 연동한 **AI 기능** 통합
- 드래그&드롭, 칸반 보드 등 **인터랙티브 UI** 구현

## 2. 학습 포인트

### 2.1 백엔드 (Python FastAPI)

| 학습 항목 | 실습 내용 | 관련 파일 |
|-----------|-----------|-----------|
| **FastAPI 기본 구조** | 라우터 분리, 의존성 주입, Pydantic 모델 | `main.py`, `routers/`, `models/` |
| **JWT 인증** | 토큰 발급/검증, Bearer 인증, 비밀번호 해싱(bcrypt) | `utils/security.py`, `services/auth_service.py` |
| **SQLite3 CRUD** | 테이블 생성, 인덱스, 파라미터 바인딩(SQL Injection 방지) | `database.py`, `services/note_service.py` |
| **CORS 설정** | 프론트엔드-백엔드 통신을 위한 CORS 미들웨어 | `main.py` |
| **AI API 연동** | Google Gemini SDK 호출, 프롬프트 엔지니어링 | `services/ai_service.py` |
| **환경변수 관리** | `.env` 파일로 민감 정보 분리 | `config.py`, `.env` |

#### 핵심 코드 패턴

```python
# 1. FastAPI 라우터 + 의존성 주입 패턴
@router.post("/register", response_model=UserResponse)
async def register(
    user_data: UserCreate,                                    # Pydantic 모델로 자동 검증
    conn: Annotated[sqlite3.Connection, Depends(get_db)]      # DB 의존성 주입
):
    return register_user(conn, user_data)

# 2. JWT 인증 미들웨어 패턴
@router.get("/me")
async def get_me(
    current_user: Annotated[dict, Depends(get_current_user)]  # 토큰에서 사용자 추출
):
    ...

# 3. SQL Injection 방지 (파라미터 바인딩)
cursor.execute("SELECT * FROM users WHERE email = ?", (email,))  # ✅ 안전
cursor.execute(f"SELECT * FROM users WHERE email = '{email}'")   # ❌ 위험
```

### 2.2 프론트엔드 (Next.js + React)

| 학습 항목 | 실습 내용 | 관련 파일 |
|-----------|-----------|-----------|
| **App Router** | 파일 기반 라우팅, 레이아웃, 페이지 구성 | `app/layout.tsx`, `app/*/page.tsx` |
| **커스텀 훅** | 인증 상태 관리(`useAuth`), 메모 CRUD(`useNotes`) | `hooks/useAuth.ts`, `hooks/useNotes.ts` |
| **API 클라이언트** | fetch 래퍼, JWT 토큰 자동 첨부, 에러 핸들링 | `lib/api.ts` |
| **Tailwind CSS** | 유틸리티 클래스, 반응형 디자인(`md:` breakpoint) | 모든 컴포넌트 |
| **드래그&드롭** | Mouse 이벤트 기반 자유 배치, HTML5 Drag API 칸반 | `FreeformView.tsx`, `KanbanView.tsx` |
| **Markdown 렌더링** | react-markdown + remark-gfm | `MarkdownRenderer.tsx` |
| **반응형 디자인** | CSS 기반 모바일/데스크톱 전환 | `FreeformView.tsx`, `KanbanView.tsx` |

#### 핵심 코드 패턴

```tsx
// 1. 커스텀 훅으로 상태 관리
const { user, login, logout } = useAuth();
const { notes, createNote, updateNote, deleteNote } = useNotes();

// 2. API 클라이언트 - JWT 자동 첨부
async function fetchWithAuth(endpoint, options) {
  const token = localStorage.getItem('access_token');
  headers['Authorization'] = `Bearer ${token}`;  // 자동 첨부
  ...
}

// 3. Tailwind CSS 반응형 디자인
<div className="block md:hidden">   {/* 모바일에서만 표시 */}
<div className="hidden md:block">   {/* 데스크톱에서만 표시 */}
```

## 4. 데이터 흐름

### 4.1 인증 플로우

```
[회원가입]
사용자 → POST /api/auth/register → 비밀번호 bcrypt 해싱 → DB 저장

[로그인]
사용자 → POST /api/auth/login → 비밀번호 검증 → JWT 토큰 발급
         ↓
사용자 ← { access_token: "eyJ..." } ← 클라이언트 localStorage 저장

[인증된 요청]
사용자 → GET /api/notes (+ Authorization: Bearer eyJ...) → 토큰 검증 → 데이터 반환
```

### 4.2 메모 CRUD 플로우

```
[생성] POST   /api/notes          → DB INSERT → 생성된 메모 반환
[조회] GET    /api/notes           → DB SELECT → 메모 목록 반환
[수정] PUT    /api/notes/{id}      → DB UPDATE → 수정된 메모 반환
[삭제] DELETE /api/notes/{id}      → DB DELETE → 성공 메시지 반환
[위치] PATCH  /api/notes/{id}/position → Freeform 좌표 저장
[칸반] PATCH  /api/notes/{id}/kanban   → 상태(plan/progress/done) 변경
```

## 6. 실행 환경

| 구분 | 요구사항 |
|------|---------|
| Node.js | 18+ |
| Python | 3.10+ |
| Gemini API Key | [Google AI Studio](https://aistudio.google.com/)에서 발급 (AI 기능용, 선택사항) |

### 빠른 시작

```bash
# 1. 백엔드 실행
cd backend
pip install -r requirements.txt
python main.py                    # http://localhost:8000 (자동 reload)

# 2. 프론트엔드 실행 (새 터미널)
cd frontend
npm install
npm run dev                       # http://localhost:3000

# 3. 브라우저에서 http://localhost:3000 접속
#    → 회원가입 → 로그인 → 메모 작성!
```
