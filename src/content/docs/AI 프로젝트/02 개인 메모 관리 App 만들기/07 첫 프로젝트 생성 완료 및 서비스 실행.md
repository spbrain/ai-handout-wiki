이 부분은 프로젝트의 완료 상태로 문서를 업데이트하는 과정입니다.

# 프로젝트 실행
모델에 따라서 첫 프로젝트가 생성이 끝이 나면 다음과 같은 상태가 있을수 있습니다.

1. 프로젝트 생성 완료 이후 아무런 Action 없음
2. 프로젝트 생성 완료 이후 프로젝트 실행 까지 수행
3. 프로젝트 생성 완료 이후 프로젝트 실행 및 Browser에서 확인까지 수행
이 부분은 명시하지 않았기 때문에 이 3가지 중에 하나로 완료될수 있습니다.

만약 아무런 Action이 없으면 명령어 창에 다음처럼 입력합니다.
"전체 프로젝트 개요와 프로젝트 실행할 수 있도로 README.md를 만들어줘"

### README.md 확인
